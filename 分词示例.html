1.1 大文件分词示例

jieba对几十G的文本的单进程分词效率不满足要求，需要并行分词

# test.txt 的格式为 label+\t+内容，例如：

1    [中国移动]您的手机已经欠费停机，请及时充值

1.1.1 切分文件

用linux 命令split 切分文件 

 #假设共15000行，每1000行生产一个文件,前缀为 input

 split -l 1000 test.txt input

会产生文件名为input01, input02， ..., input15

import jieba
with open('conf/userdict.txt', 'r') as aa:
    for a in aa:
        a = a.strip()
        jieba.add_word(a)

1.1.3 并行分词

#coding:utf-8
import time
import jieba
from multiprocessing import Pool
import os

begin = time.time()
n = 0

def p(i):
    fin = open('input'+str(i).zfill(2),'r')
    fout = open('out'+str(i).zfill(2),'w')
    n = 0
    for line in fin:
        content = line.strip().split('\t')[1]
        label = line.strip().split('\t')[0]
        words = ' '.join(jieba.cut(content, cut_all=False))

        print >> fout, "__label__%s\t%s" % (label.encode('utf8'),words.encode('utf8'))
    fin.close()
    fout.close()

# 开启17个进程
pool = Pool(processes=17)
print(' begin...')

for i in range(17):
    result = pool.apply_async(p, (i,))
pool.close()
pool.join()

end = time.time()
print('time_read', (end-begin)/60)
# 对所有的输出结果进行合并
os.system("sh save.sh")
